## 🎯 Como Usar (Resumo Rápido)

### Windows
```
1. Duplo clique em: App_Treinos.bat
2. Pronto! ✅
```

### macOS
```bash
1. chmod +x App_Treinos.command
2. Duplo clique em: App_Treinos.command
3. (Primeira vez: permitir em Segurança se necessário)
```

### Linux
```bash
1. chmod +x linux/app_treinos_launcher.sh
2. cp linux/App_Treinos.desktop ~/.local/share/applications/
3. Procure "App Treinos" no menu de aplicativos
```
